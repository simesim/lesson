1. Что такое терминал?

Терминал — это окно, где пользователь пишет команды текстом. В Windows чаще всего используют:

CMD

Старая командная строка Windows. Команды: dir, cd, cls, ipconfig.

PowerShell

Более мощная оболочка. Команды: Get-ChildItem, Get-Process, Get-Date.

Windows Terminal

Современное приложение, где можно открывать CMD, PowerShell и WSL во вкладках.

Команда обычно состоит из названия и дополнительных параметров. Например: dir /p, где dir — команда, а /p — параметр.

2. Полезные команды Windows

Нажимай на команду, чтобы раскрыть объяснение. Это удобно делать с телефона.

dir CMD

Показывает файлы и папки в текущей папке.

C:\Users\Student> dir
cd CMD

Переход в другую папку. cd .. — подняться на уровень выше.

cd Desktop
cd ..
cls CMD

Очищает экран терминала.

mkdir CMD

Создаёт новую папку.

mkdir lesson
rmdir CMD

Удаляет пустую папку. Осторожно: удаление нельзя делать случайно.

rmdir lesson
copy CMD

Копирует файл.

copy notes.txt backup.txt
move CMD

Перемещает или переименовывает файл.

move old.txt new.txt
del CMD

Удаляет файл. Команда опасная: лучше сначала проверить, что удаляешь.

del trash.txt
type CMD

Показывает содержимое текстового файла.

type readme.txt
echo CMD

Выводит текст или записывает текст в файл.

echo Привет!
echo Hello > hello.txt
help CMD

Показывает список команд CMD или справку по команде.

help
help dir
whoami CMD/PS

Показывает имя текущего пользователя.

ipconfig CMD

Показывает информацию о сетевых адаптерах и IP-адресах.

ping CMD

Проверяет, отвечает ли другой компьютер или сайт.

ping ya.ru
tasklist CMD

Показывает запущенные процессы.

Get-ChildItem PowerShell

PowerShell-аналог команды dir. Также можно писать коротко: gci.

Set-Location PowerShell

PowerShell-аналог команды cd. Коротко: sl.

Get-Process PowerShell

Показывает запущенные процессы.

Get-Date PowerShell

Показывает дату и время.

Clear-Host PowerShell

Очищает экран PowerShell. Коротко: cls.

3. Мини-тренажёр терминала

Введи одну из команд: dir, cd desktop, cd .., cls, whoami, ipconfig, echo привет, mkdir test, ping ya.ru, help.

Windows Terminal — учебная симуляция
C:\Users\Student> готов к командам

Это симулятор. Он ничего не удаляет и не меняет на устройстве.

6. Смешная команда: танцующий попугай

В реальном терминале можно выводить ASCII-art через echo. Здесь — безопасная демонстрация.

(\_/)
( •_•)
/ >🦜 танцует терминальный попугай
echo        __
echo       (o )___
echo        ( ._> /
echo        `---'   terminal parrot!
